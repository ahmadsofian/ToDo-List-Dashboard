# ToDo-List-Dashboard

A zero-dependency, local-first task management dashboard built with pure HTML, CSS, and JavaScript. 

Instead of a standard top-to-bottom list, this application features a split-view architecture inspired by professional business intelligence tools. It tracks task completion rates, visualizes priority loads, and generates real-time performance metrics without requiring an external database or backend server.

### Key Features
* **KPI Tracking & Analytics:** Real-time calculation of completion percentages and pending workloads.
* **Dynamic Data Visualization:** Custom-built bar charts and progress gauges that scale automatically based on task metrics.
* **Privacy by Design (Local Storage):** Zero data is sent to the cloud. All tasks and metrics are stored locally in the browser, ensuring complete user privacy and compliance.
* **Zero Dependency:** No external libraries, frameworks, or database setups required.

### How to Use
1. Clone the repository or download the `index.html` file.
2. Double-click `index.html` to open it in any modern web browser.
3. Your data will automatically persist across sessions using your browser's local storage.

### Tech Stack
* **UI/UX:** Vanilla HTML5 & CSS3 (Flexbox/Grid for layout)
* **Logic & Analytics:** Vanilla JavaScript
* **Storage:** Browser `localStorage` API
